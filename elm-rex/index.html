Currying

Currying is the process of translating the evaluation of a function that takes multiple arguments into evaluating a sequence of functions, each with a single argument. Source Wikipedia

Currying

          
            add : Int -> Int -> Int
            add number1 number2 = number1 + number2

            -- Let's curry it!

            add5 : Int -> Int
            add5 value = value + 5

            -- Let's use it now!

            marks : List Int
            bumpedMarks = List.map add5 marks

            add5 = (+) 5
            bumpedMarks = List.map ( (+) 5 ) marks

Currying - real example

          
            coeff : Float
            coeff = List.repeat precision "0"
              |> String.join ""
              |> (++) "1"
              |> String.toFloat
              |> Maybe.withDefault 1.0

A sound type-system

  • Strong: the type is always known
  • Structural: the type is defined by it's structure
  • No null: places where values can be missing are known
  • Implicit OR explicit: whatever fits the local needs

Records

          
            type alias Person = { name: String, dob: Date }
            -- Explicit typing
            getAge : Person -> Int
            -- Implicit typing
            getAge { dob } = ...
            -- Semi-implicit typing
            getAge : { whatever | dob: Date }
